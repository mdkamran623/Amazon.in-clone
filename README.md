# Amazon.in Clone 🛒

A simple front-end clone of Amazon.in homepage built with HTML, CSS, and JavaScript.

## Features

- Responsive navigation bar
- Auto-sliding image banner with manual controls
- Product cards with prices and deals
- Mobile-friendly design
- 4 product categories

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons

## Project Structure

├── style.css # Stylesheet
